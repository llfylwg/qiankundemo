# 保卖商家后台（企业版）

## 代码拉取

```
git clone http://gitlab.zhuanspirit.com/zz-fe-u/BM-Biz-Manage.git --recursive
```

## 依赖安装

```
npm install -g @vue/cli
npm install
```

## 本地运行

```
npm run dev
```

### 本地接口 mock

```
npm run server
```

### 打包

```
npm run build
```
